Overview

This project is a Tic-Tac-Toe game powered by AI, featuring two algorithms: Minimax and Alpha-Beta pruning. Players can compete against the AI or each other.

Installation

To install the project, clone the repository and install the required dependencies:

git clone https://github.com/yourusername/tic-tac-toe-ai.git
cd tic-tac-toe-ai
pip install -r requirements.txt

Usage

Run the application using the following command:

python app.py

Then open http://127.0.0.1:5000 in your browser to play the game.

API

The application exposes the following API endpoints:

  • GET / - Serves the game UI
  • POST /api/move - Takes the current board and algorithm, returns the AI's chosen move and stats
  • POST /api/reset - Returns a fresh board
